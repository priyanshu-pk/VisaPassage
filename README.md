# VisaPassage
Small Scale Project 
A webiste which can be used to contact an agent for different services like 
tourist visa / buisness visa etc. 





**🛠 Tech Stack Used**
Frontend: HTML, CSS, JavaScript

Styling: CSS (Flexbox, Grid, Animations)

Interactivity: Vanilla JavaScript (DOM Manipulation, Fetch API)

Icons & UI Enhancements: Font Awesome, Google Fonts
